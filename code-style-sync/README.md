- # 📂 你的私有算法与规范宝库

  这里是 AI 的“外挂记忆区”。要让这个技能真正跑起来，你需要按照以下方式初始化你的参考资料：

  ### 1. 初始化方式 (二选一)
  - **手动搬运：** 把你平时在 AcWing、LeetCode 或本地写得最漂亮的 `Dijkstra.cpp`, `Trie.cpp` 等文件重命名后扔进来。
  - **自动采集：** 在 IDE 侧边栏对 AI 说：“*@SKILL.md 学习我当前项目中 src 目录下的代码，把里面的通用算法和命名习惯存入你的参考库。*”

  ### 2. 推荐文件命名规范
  - `strict_rules.md`: 存放你对变量命名、缩进、注释的死规定。
  - `algo_templates.cpp`: 存放你经过打磨的、极致效率的算法底座。
  - `boilerplate.md`: 存放项目通用的文件头部、错误处理、日志打印模板。

  ### 3. 注意事项
  AI 会根据你的代码风格自动迭代。如果你发现 AI 学习到了错误的风格，请直接修改此目录下的文件，AI 会在下一次任务中立即更正。

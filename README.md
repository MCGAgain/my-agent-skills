# 🚀 My AI Toolset: Skills & Prompts

这是一个cs大一菜鸟随便用ai写的一段，希望没人看

本项目致力于解决 AI 辅助编程中的“代码风格割裂”痛点。不要让 AI 改变你的习惯，而是让 AI 适应你的风格。

## 📂 核心架构与目录

### 1. 🤖 Agent Skills
**目录:** `/code-style-sync` 等包含 `SKILL.md` 与 `references/` 的独立文件夹。
**适用环境:** 具备 Agent 能力与本地文件读取权限的 IDE (Cursor, Windsurf, Cline 等)。
**原理解析:** 采用 **RAG（检索增强生成）** 模式。以文件目录作为 AI 的“外挂大脑”。触发指令后，Agent 会主动读取用户存放在 `references` 中的底层模板和硬性排版规范，实现全自动的风格同步，零对话摩擦力。

### 2. 🧰 Prompts
**目录:** `/prompts` 下的独立 `.md` 文件。
**适用环境:** 网页端对话大模型 (Gemini, ChatGPT, Claude) 或任何不具备本地读取权限的临时环境（如在图书馆刷题）。
**原理解析:** 采用 **Few-Shot Prompting (少样本提示)** 的结构化填空模板。即使在无法直接读取本地代码的网页端，也能通过标准化的输入格式，瞬间给 AI 注入你的极客风格与算法偏好。

## 🚀 快速上手 (Usage)

### 场景 A：在本地 IDE 中无缝编码 (推荐)
1. 将本仓库 clone 到本地计算机。
2. 在 Cursor 或 Windsurf 的对话侧边栏，键入 `@` 唤出上下文选择器。
3. 选中并引入 `code-style-sync/SKILL.md` 文件。
4. 附上你的开发需求，例如：“*用里面的规范帮我写一个线段树模板。*”
> *AI 将静默读取同目录下的参考文件，并直接输出带有你个人印记的代码。*

### 场景 B：在网页端探讨算法与题解
1. 打开本仓库的 `prompts/style_cloner_template.md` 文件。
2. 复制文件内的所有结构化文本。
3. 按照模板内的提示，在【我的参考代码区】粘贴一段你以前在 AcWing 或本地写过的得意代码，并在【我的新任务区】写下当前需求。
4. 发送给网页端 AI 即可获得完美模仿的结果。

## 🚀 进阶技巧：开启“主动学习”模式

本项目的 Skill 具备 `Self-Evolution` 逻辑。你可以通过以下指令让 AI 帮你维护知识库：

- **反向同步：** “*@SKILL.md 扫描我这个文件的逻辑，如果比你现有的模板更好，请帮我同步到 references。*”
- **风格体检：** “*@SKILL.md 对比当前代码与 references 里的规范，指出我不一致的地方。*”
